General rules

The following rules apply to all servers:

General rules of conduct:

  • Hate-Speech/racism and other forms of discrimination is forbidden, it will result in a comms ban or ban.
  • Harassment and bullying of any kind is forbidden, it will result in a comms ban or ban.
  • Sexism is forbidden, it will result in a comms ban or ban.
  • You can appeal a ban here.

Game specific rules

  • Admins and moderators are sovereign. They decide what is allowed and what is not allowed. You can report abuses on Discord.
  • Server squatting is forbidden, it will result in a ban.
  • Any form of hacking is forbidden, it will result in a ban.
  • Playing music or other sounds on the server is forbidden, it will result in a comms ban.
  • Abusing the !calladmin command is forbidden, it will result in a ban.

General commands:

Commands can be typed direcly in chat
  • !report: Report a player to the admins, if none are online.
  • !rank: Show your rank on the server.
  • !calladmin: Call an admin on Discord.
  • !sl: Lists all Tensor servers.
  • !votekick: Vote to kick a player.
  • !voteban: Vote to ban a player.

The following commands only work on the AWP servers:

Those commands can be typed direcly in chat

  • !rs: Reset your score on the server.

Surf Servers Rules

The following rules only apply to the Surf servers:

  • The use of "Surf Bots" is forbidden, and will result in a ban.
  • The use of checkpoints/stages exploits is forbidden, and will result in a ban.

The following commands only work on the Surf servers:

Those commands can be typed direcly in chat
  • !r: Reset your run.
  • !s "stagenumber": Teleport to the stage.
  • !end: Teleport to the end.
  • !style: Set your surfing style.
  • !help: Full list of commands.
  • !rtv: Start a rock the vote. If enough people vote, the map will change.

The following commands only work on the Retakes servers:
